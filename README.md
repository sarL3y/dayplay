## Getting Started
- Fork it
- From root: `npm install`
- `cd client/` and `npm install` again
- Create MongoDB cluster and add connection string to `MONGODB_URI=` env variable
- From root: `npm start`
- `cd client/` and `npm start` again (or run concurrently)
